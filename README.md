Isso é quase uma brincadeira, de alguém que trabalha com mídia DOOH e vê muita planilha pela frente! 

O sistema é construído todo com IA, sem conhecimento de programação: um sistema de gestão, que além de constroles, acabou de ganhar também, um módulo que programação de campanha, ou seja, ele também virou um AdSever para os painéis, algo parecido com o XIBO.

Ele é feito em Laravel 12 + Filament 5 e roda em uma hospedagem compartilhada.
